# Viovox static site for GitHub Pages

This archive is ready for GitHub Pages.

## Structure

- `index.html` — main page, must be in the repository root
- `styles.css` — all site styles
- `script.js` — small rendering script for cards and current year
- `assets/viovox-symbol.svg` — logo/favicon
- `CNAME` — custom domain `viovox.net`
- `.nojekyll` — prevents GitHub Pages from processing the site with Jekyll

## How to publish

1. Create or open your GitHub Pages repository.
2. Upload all files from this ZIP to the repository root, not inside an extra folder.
3. In GitHub: Settings → Pages → Deploy from branch → `main` / root.
4. If you use `viovox.net`, keep the `CNAME` file.
